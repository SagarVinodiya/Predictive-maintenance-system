# Predictive Maintenance System

AI-powered predictive maintenance platform for industrial equipment.

## Features
- Equipment health monitoring
- AI predictions with multiple ML algorithms
- Voice assistant (EVA)
- Interactive dashboard

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR-USERNAME/predictive-maintenance.git
cd predictive-maintenance
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the dashboard:
```bash
streamlit run app.py
```

4. Open landing page:
Open `index.html` in your browser

## Usage
1. Upload your equipment CSV data
2. Configure target and feature columns
3. Train AI models
4. Search equipment and get predictions

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Streamlit
- ML: Scikit-learn, Random Forest, SVM, KNN
